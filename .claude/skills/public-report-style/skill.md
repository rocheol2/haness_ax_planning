# Public Report Style — 개조식·공공기관 어투 작성 가이드

report-writer 에이전트의 기획 보고서 문체 품질을 표준화하는 가이드.

## 개조식 기호 체계

| 계층 | 기호 | 용도 | 들여쓰기 |
|------|------|------|---------|
| 1 (대항목) | □ | 절 안의 핵심 골자 | 없음 |
| 2 (중항목) | ○ | 대항목의 구체화 | 1칸 |
| 3 (소항목) | - | 세부 내용·근거 | 2칸 |
| 4 (세부) | · | 예시·부연 | 3칸 |
| 참고 | ※ | 주의·참고 사항 | 해당 항목 아래 |
| 괄호 표제 | (일시), (장소) | 개요 항목의 머리말 | □ 뒤 |

**예시**:

    □ 전사 AX 역량 격차 심화에 따른 체계적 교육 필요
     ○ '25년 사내 설문 결과 생성형 AI 활용률 부서 간 최대 4배 차이
      - 활용 상위: IT·기획(78%) / 하위: 현장·지원(19%)
     ※ 세부 결과는 붙임 1 참조

## 어미 규칙 (공공기관 어투)

| 용도 | 어미 | 예시 |
|------|------|------|
| 현황·사실 | ~임, ~함, ~됨 | 「참여 수요가 증가하고 있음」 |
| 의지·계획 | ~하고자 함, ~할 계획임, ~추진 | 「11월 중 개최하고자 함」 |
| 필요·당위 | ~필요, ~요구됨 | 「전문 강사 섭외 필요」 |
| 예정 | ~예정 | 「10월 말 홍보 개시 예정」 |
| 명사형 종결 | ~개최, ~운영, ~실시 | 「분기별 1회 실시」 |

**금지 사항**:
- 서술형 종결(~합니다, ~입니다, ~한다) 사용 금지
- 구어체·감성적 수식어(매우, 정말, 꼭) 사용 금지
- 영어 약어 첫 등장 시 한글 병기: 「AX(AI 전환)」

## 서술형 → 개조식 변환 사전

| 서술형 (금지) | 개조식 (권장) |
|--------------|--------------|
| 최근 기업들이 AI 도입을 서두르고 있습니다 | □ 기업의 AI 도입 가속화 추세 |
| 이 워크샵을 통해 실무 역량을 키우고자 합니다 | □ 실습 중심 워크샵을 통한 실무 역량 제고 |
| 예산은 총 2천만원이 필요할 것으로 예상됩니다 | □ (소요 예산) 총 20,000천원 |
| 참가자들의 만족도가 높을 것으로 기대됩니다 | □ 참가자 만족도 4.5점 이상(5점 척도) 달성 목표 |
| 강사 섭외가 어려울 수도 있습니다 | ※ 강사 섭외 지연 시 사내 전문가 대체 운영 |

## 압축 원칙

1. **한 항목 한 내용**: 두 가지 내용은 항목을 분리한다
2. **2줄 이내**: 항목이 2줄을 넘으면 하위 항목으로 분리한다
3. **조사 생략**: 「~을/를 위한」 → 「~ 위한」, 불필요한 조사는 삭제한다
4. **수치 근거 괄호 병기**: 「규모 100명 내외(전년 87명 대비 확대)」
5. **명사 종결 우선**: 동사보다 명사형으로 끝나면 더 압축된다

## 숫자·날짜·단위 표기

| 항목 | 표기 규칙 | 예시 |
|------|----------|------|
| 날짜 | '년도 축약 + 마침표 | '26. 11. 12.(목) |
| 시간 | 24시간제 물결 | 14:00~17:00 |
| 금액 | 천원 단위, 콤마 | 20,000천원 |
| 인원 | 명 + 내외/이상 | 50명 내외 |
| 비율 | % + 기준 병기 | 78%('25. 6월 기준) |

## 제목·개요 관행

- 보고서 제목: 「○○○ 개최 계획(안)」 — 미확정 계획은 반드시 「(안)」 표기
- 개요 항목 순서: (행사명) → (일시) → (장소) → (대상) → (주최·주관) 고정
- 붙임 문서: 「붙임 1. 세부 프로그램 1부.」 형식, 마지막에 「끝.」

## 품질 체크리스트

| 항목 | 기준 |
|------|------|
| 어미 통일 | 서술형 종결 0건 |
| 기호 계층 | □→○→-→· 순서 일탈 없음 |
| 항목 길이 | 2줄 초과 항목 없음 |
| 수치 근거 | 주요 수치에 출처·기준 병기 |
| (안) 표기 | 미확정 항목 전부 표기 |
| 날짜 표기 | '00. 00. 00.(요일) 형식 통일 |
