# Executive QA Patterns — 상급자 예상 질의·실행가능성 점검 패턴

analyst / executive-summarizer 에이전트의 상급자 관점 검증을 표준화하는 패턴.

## 결재권자 질의 5대 축

모든 기획 보고서는 다음 5개 축의 질문을 통과해야 한다:

| 축 | 대표 질문 | 답이 있어야 할 위치 |
|----|----------|-------------------|
| **Why now** | 왜 지금 해야 하는가? 내년에 하면 안 되나? | Ⅰ. 추진 배경 |
| **Why this** | 왜 이 방식(포럼/워크샵/교육)인가? 대안은 검토했나? | Ⅱ. 목적, Ⅳ. 세부 계획 |
| **How much** | 예산 산출 근거는? 더 싸게 할 수 없나? | Ⅵ. 소요 예산 |
| **What if** | 강사 섭외 실패하면? 참가 저조하면? | 리스크 대응(본문 또는 Q&A) |
| **So what** | 하고 나면 뭐가 달라지나? 어떻게 측정하나? | Ⅶ. 기대 효과 |

## 행사 유형별 단골 질의

### 포럼·세미나
1. 외부 공개 행사인가, 사내 행사인가 — 보안·홍보 방침은?
2. 연사 라인업이 참가자를 모을 수준인가?
3. 유사 행사와 차별점은 무엇인가?
4. 일회성인가, 정례화 계획이 있는가?

### 워크샵
1. 실습 산출물이 실제 업무에 연결되는가?
2. 참가자 수준 편차는 어떻게 관리하나?
3. 실습 환경(계정, 장비, 보안) 준비는?
4. 사후 팔로업 계획이 있는가?

### 교육
1. 기존 교육과 중복되지 않는가?
2. 교육 대상 선발 기준은?
3. 업무 공백(교육 시간) 부담은 어떻게 하나?
4. 교육 효과를 어떻게 평가하나 (수료 기준, 현업 적용률)?

## 실행가능성 점검 프레임워크 (BSTR)

| 요소 | 점검 질문 | 판정 기준 |
|------|----------|----------|
| **B**udget (예산) | 모든 항목에 산출 근거(단가×수량)가 있는가 | 근거 없는 항목 0건 |
| **S**chedule (일정) | D-day 역산으로 리드타임이 확보되는가 | 섭외 D-60, 홍보 D-30, 장소 D-45 이상 |
| **T**eam (인력) | 과업별 담당이 지정 가능한 규모인가 | 담당 미지정 과업 0건 |
| **R**isk (리스크) | 상위 3개 리스크에 Plan B가 있는가 | 대응안 없는 핵심 리스크 0건 |

### 리드타임 참고 기준

| 과업 | 최소 리드타임 | 비고 |
|------|-------------|------|
| 외부 연사 섭외 | D-60 | 유명 연사는 D-90 |
| 장소 대관 | D-45 | 성수기(10~11월)는 D-90 |
| 홍보·모집 | D-30 | 사내 공지 기준, 외부 모집은 D-45 |
| 교재·자료 제작 | D-14 | 인쇄 포함 시 D-21 |
| 최종 리허설 | D-3 | 실습형은 환경 점검 포함 |

## Q&A 작성 규칙

1. **질문은 깐깐하게**: 우호적 질문이 아니라 결재를 보류시킬 수 있는 질문을 상정한다
2. **답변은 근거로**: "잘 준비하겠음"이 아니라 수치·사례·대안으로 답한다
3. **본문 반영 우선**: 좋은 답변은 Q&A에만 두지 말고 본문에 선제 반영하도록 report-writer에게 피드백한다
4. **답 못하는 질문은 정직하게**: 근거가 없으면 "추가 확인 필요"로 표기하고 확인 방법을 제시한다

## 검증 심각도 판정 기준

| 심각도 | 기준 | 예시 |
|--------|------|------|
| [필수 수정] | 결재 반려 사유가 되는 결함 | 예산 산출 근거 누락, 수치 불일치, 일정 역산 불가능 |
| [권장 수정] | 질문을 유발하는 약점 | 기대효과 측정 방법 모호, 대안 검토 흔적 없음 |
| [참고 사항] | 완성도를 높이는 개선점 | 표현 다듬기, 붙임 자료 보강 제안 |

## 품질 체크리스트

| 항목 | 기준 |
|------|------|
| 질의 5대 축 | Why now / Why this / How much / What if / So what 전부 커버 |
| 질의 수 | 행사 유형별 단골 질의 포함 8~12개 |
| BSTR 점검 | 4개 요소 전부 판정 완료 |
| 답변 근거율 | 근거 없는 답변 0건 (미확인은 명시) |
| 본문 반영 피드백 | 핵심 Q&A의 본문 선반영 여부 확인 |
